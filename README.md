# KMUpower

## Übersicht

KMUpower ist eine professionelle Website, die auf Jekyll und GitHub Pages basiert. Die Website bietet eine optimale Benutzererfahrung mit responsivem Design, Barrierefreiheit und SEO-Optimierung.

## Features

- Responsive Design für alle Geräte (Desktop, Tablet, Mobile)
- Barrierefreie Komponenten nach WCAG-Richtlinien
- Optimierte SEO mit strukturierten Daten
- Semantisches HTML5 und modernes CSS
- Optimierte Performance und Ladezeiten

## Projekt-Struktur

- `_config.yml`: Hauptkonfigurationsdatei für Jekyll
- `_layouts/`: Enthält Layoutvorlagen für die Website
- `_includes/`: Wiederverwendbare Komponenten (Karten, Formulare, etc.)
- `assets/`: Stylesheets, Bilder und andere statische Assets
- `*.md`: Markdown-Dateien für die Hauptseiten der Website

## Entwicklung

Um das Projekt lokal zu entwickeln:

1. Ruby und Bundler installieren
2. Im Projektverzeichnis `bundle install` ausführen
3. Server starten mit `bundle exec jekyll serve`
4. Website unter `http://localhost:4000` aufrufen
